# 自己推薦書に記載可能な説明文（日本語）

学部で情報工学を学ぶ一方で、近年は生物統計学および因果推論に強い関心を持ち、特に実世界データにおける生存時間解析上のバイアスに注目して自主的に学習を進めてきた。単に統計手法を実装するだけではなく、どのような研究設計や変数の扱いが推定量の解釈を変えてしまうのかを理解することが重要であると考え、簡単なシミュレーションを自作し、GitHub 上で再現可能な形に整理した。

具体的には、propensity score model の誤指定が inverse probability weighting にどのような影響を及ぼすか、landmark analysis が immortal time bias を避ける一方で survivor selection を導入し得ること、さらに treatment を baseline 固定変数として扱うのではなく time-dependent Cox model により時間順序を明示的に扱う必要があることを、小規模なコード例によって比較・検討した。これらの作業を通じて、統計解析とは単なる計算ではなく、estimand、時間順序、交絡構造、選択バイアスを意識しながら設計すべき方法論的営みであるという理解が深まった。

今後は、このような基礎的理解を出発点として、実世界データに対する因果推論、time-varying confounding、marginal structural model など、より高度な生物統計学の方法へ発展的に取り組みたいと考えている。
